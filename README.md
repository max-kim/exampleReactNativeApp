# Example React Native App

## Installation

You can install 'sORM' with these commands:
```bash
    $ git clone https://github.com/max-kim/exampleReactNativeApp.git
    $ cd exampleReactNativeApp
    $ npm install
    $ cd ./ios && pod install && cd ..
    $ react-native start
    $ react-native run-android
```

## Usage

```bash
    $ react-native start
    $ react-native run-android
```
